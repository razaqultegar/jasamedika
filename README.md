    ```bash
    composer install
    npm install
    npm run dev
    ```

    ```bash
    cp .env.example .env
    ```

    ```bash
    php artisan key:generate
    ```

5. **Set up databases**:

    ```bash
    php artisan migrate:fresh --seed
    ```

6. **Start the development server**:

    ```bash
    php artisan serve
    ```
